STATA/LaTex

templates to make STATA and LaTex play nice and look good

Download this project as a .zip file Download this project as a tar.gz file

Features/Goals

Good Looking

STATA's defaults are often ... less than beautiful

Ready-to-use LaTeX text with commands/files generated from within STATA

Because lazy is better, and spending time on writing and re-writing LaTeX feels wasteful

Templates

Summary Statistics, Regressions, Displaying Code

Scatter Plots, Categorical Plots, Maps
